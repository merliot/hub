Requirement: latest Go installed.

Pull the Merliot Hub source code from github and compile and run the hub from the command line:

$ git clone https://github.com/merliot/hub.git
$ cd hub
$ go run ./cmd

Browse to http://localhost:8000 to view hub.

If you're prompted for user/passwd, use your $USER for the user. You can override with environment variables:

$ USER=xxx PASSWD=yyy go run ./cmd
