FREQUENTLY ASKED QUESTIONS

Can web crawlers "see" the hub?

No, if they're following the rules. The hub has a robots.txt file with

User-agent: *
Disallow: /

And has the meta tag:

<meta name="robots" content="noindex, nofollow">

What is the hub's referrer policy?

No referrer data is sent when someone navigates from the hub to other web sites. The "no-referrer" policy is set in the meta tag:

<meta name="referrer" content="no-referrer">
